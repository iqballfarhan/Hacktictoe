🎮 Hacktictoe
Hacktictoe adalah permainan Tic Tac Toe berbasis web yang interaktif dan menyenangkan. Dibangun menggunakan HTML, CSS, dan JavaScript, proyek ini memungkinkan pengguna untuk bermain melawan komputer dengan antarmuka yang sederhana namun menarik.

🧩 Fitur
Antarmuka Pengguna Interaktif: Desain yang responsif dan intuitif untuk pengalaman bermain yang optimal.

Permainan Melawan Komputer: Algoritma sederhana memungkinkan pengguna bermain melawan AI dasar.

Penyimpanan Histori Permainan: Riwayat permainan disimpan menggunakan localStorage, memungkinkan pengguna melihat hasil permainan sebelumnya.

Efek Suara: Menambahkan elemen audio untuk meningkatkan pengalaman bermain.

🚀 Demo Langsung
Anda dapat mencoba permainan ini secara langsung dengan membuka file index.html di browser Anda atau mengaksesnya melalui GitHub Pages jika telah di-deploy.
